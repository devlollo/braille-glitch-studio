# legacy

Verbatim backups of the original three scripts, kept before the restructure into
the `braille_glitch` package (July 2026). They still run on their own from this
folder: `python3 legacy/braille_glitch_studio.py` is the old self-contained live
studio, and `glitch_core.py` + `video_glitch.py` are the old engine/video pair.
